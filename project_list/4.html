Chatting App





express와 socket.io를 이용하여 채팅서버를 만들어보았습니다.
디자인에는 부트스트랩을 사용하였고 채팅서버로 접속 시 pug로 만든 페이지를 클라이언트로 전송하도록 하였습니다. 사용자는 채팅과 닉네임 변경이 가능하며 채팅방 참여, 종료, 닉네임 변경 시 채팅 로그에 알림이 뜨도록 하였습니다.





구현한 기능

  • 채팅서버에 접속 시 익명1, 익명2, ... 로 닉네임 부여
  • 채팅방 참여, 떠남 채팅로그에 알림표시
  • 닉네임 변경 시 채팅로그에 알림표시
  • 접속자 간의 채팅 가능

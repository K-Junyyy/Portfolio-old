네이버 로그인 페이지 만들기





실제 네이버 로그인 페이지의 경우 화면의 가로길이가 770px정도를 넘어서면 UI들의 크기가 고정되고 IP 보안 표시, 맨 하단 저작권 표시가 나타나고 언어 선택 박스가 살짝 이동하는 것을 확인하였습니다.

media 쿼리를 사용해서 가로길이가 800px를 넘어서면 UI 크기 고정, IP 보안, 저작권 표시를 할 수 있도록 하였습니다. 추후 JavaScript와 MongoDB를 사용하여 여러 가지 기능(유효성 검사, 로그인 성공&실패, 언어 변경 등등)들을 구현해볼 예정입니다.

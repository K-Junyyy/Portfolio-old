네이버 로그인 페이지 만들기





실제 네이버 로그인 페이지의 경우 화면의 가로길이가 770px정도를 넘어서면 UI들의 크기가 고정되고 IP 보안 표시, 맨 하단 저작권 표시가 나타나고 언어 선택 박스가 살짝 이동하는 것을 확인하였습니다.

그리고 media 쿼리를 사용해서 가로길이가 800px를 넘어서면 UI 크기 고정, IP 보안, 저작권 표시를 할 수 있도록 하였으며, 회원가입 폼을 제작하여 로그인페이지에서 회원가입 탭을 클릭 시 해당 페이지로 링크되도록 하였습니다.

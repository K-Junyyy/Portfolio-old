공공데이터API를 이용한

버스정보조회 APP Service




본 프로젝트는 "클라우드웹서비스" 강의에서 진행했던 프로젝트 입니다.

CSS지식이 없어서 디자인은 꾸지미 못하였으며,
GET&POST 부분을 잘 처리하지 못하여 추후 해결할 예정입니다.



공공데이터 API를 통해서 받아온 XML 정보들이며

파이썬으로 아래와 같이 필요한 정보들을 출력해보았습니다.



해당 데이터들은 버스 노선번호가 '721번' 일때의 정보들입니다.

현재는 CSS, Javascript, node.js 지식이 있는 상태이므로 node로 localhost에서 서버를 열고 mongoDB에 버스 정보들을 저장한 뒤 서비스가 온전히 작동하도록 개발을 진행할 예정입니다. 구현할 기능은 아래와 같습니다.

  • 버스노선 검색시 해당 노선이 지나는 정류소 목록
  • 정류소 검색시 해당 해당 정류소에 도착하는 버스들의 시간
  • 실시간 해당 노선의 버스 위치
